3 e - Inserindo um cliente já cadastrado

Caso o cliente já tenha cadastro é só fazer a busca clicando na lupa:

Aparecerá uma tela para “LOCALIZAR CLIENTE”:

Quando terminar de digitar o nome, aperte ENTER que a busca retornará os nomes cadastrados:

Aperte ENTER novamente para inserir essa cliente no pedido.

Para fazer a busca do nome do cliente podemos fazer como na imagem acima e buscar por parte do nome. Por exemplo, para a cliente “Maria das Graças Souza Duarte” podemos buscar por “GRACA DUARTE” para encontrá-la mais facilmente.
Também podemos buscar pelo nome completo “MARIA DAS GRACAS SOUZA DUARTE”. Caso não tenha certeza do nome, posso buscar por “MARIA”, o que retornará todas as clientes de nome Maria do local.
